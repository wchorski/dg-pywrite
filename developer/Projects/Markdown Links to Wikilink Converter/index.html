A NodeJS tool that converts markdown links to wikilinks and vice versa. Helpful for note taking apps like Obsidian.md

I'll be using Markdown Wikilink Example Note as an example to test this script.

Features

  • Leverage Javascript to consistently convert links via encodeURIComponent() & decodeURIComponent()
  • Automatically handle non conventional URLs that may include special characters like (, )
  • Leave external links as markdown links.
  • Run on single file, selected directory, or whole vault
  • Support for alias on wikilinks or markdown links

Note apps like Obsidian.md do not allow characters # ^ [ ] | in their filename. This converter assumes you do not use any of these characters in your filenames

findUrlsAndWriteToFile('./exampleFolder/note.md', 'wikilink');
# or
readDirectory('./exampleFolder', 'markdownLink')
