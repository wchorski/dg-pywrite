## Settings
- Finder, show whole file path in header
```shell
defaults write com.apple.finder _FXShowPosixPathInTitle -bool true
killall Finder
```

- in Finder, sort folders first
```shell
defaults write com.apple.finder _FXSortFoldersFirst -bool true
killall Finder
```
